PROYECTO FINAL INTEGRADOR

'BOCADO'
Aplicación mobile desarrollada en Android Studio orientada a la gestión de pedidos en restaurantes mediante código QR.

DESCRIPCIÓN
La aplicación permite a los clientes visualizar el menú digital de un restaurante, seleccionar platos, realizar pedidos y efectuar pagos desde el dispositivo móvil.
El sistema centraliza el flujo de trabajo del restaurante de manera automática.

FUNCIONALIDADES PRINCIPALES
* Escaneo de QR para acceso al menú
* Visualización de platos disponibles
* Gestión de carrito y pedidos
* Simulación/procesamiento de pagos

TECNOLOGÍAS UTILZADAS
* Kotlin
* Android Studio
* Jetpack Compose / XML
* API de pagos (simulada o integrada)
